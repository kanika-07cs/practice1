Form field and Validation

to set email address,input type="email"->this includes @ and letters followed by it and required->this field shouldn't be left empty
to set age,input type="number" and required it also has increment and decrement option button
min and max value to set range of input
to include dob,input type="date", by default dd-mm-yyyy
to set range slider,input type="range" give min and max and step->which increments by its count
to include time->input type="time"
to include url link->input type="url", it accepts only with correct protocol
to include week->input type="week", mention which week and its year
to search->input type= "search", the only difference b/w text type and search type is that we can clear input content in search type
